# Version Status

Version: `v0.1.0-local-acceptance`

Status: local acceptance build

## Currently Supported

- mock mode
- local_json mode
- four-level comment tree export
- CSV / XLSX export
- text cleaning
- quality_report
- audit.jsonl
- one-click acceptance script

## Currently Not Supported

- real platform requests
- automatic login
- captcha bypass
- rate-limit evasion
- video download
- profile video collection
- keyword search collection

## Safety Boundary

This version is an offline local research acceptance build. It does not include
real Douyin requests, private endpoint addresses, automatic login, captcha
bypass, rate-limit evasion, or inaccessible-data collection logic.
